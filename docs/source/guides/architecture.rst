Architecture
============

Project Structure
-----------------

The repository is organized by feature area:

- ``src/config``: configuration defaults and helpers
- ``src/nema_quant``: single-run analysis
- ``src/nema_quant_iter``: iteration-based analysis
- ``src/nema_merge``: merge and report across experiments
- ``data``: sample data and assets for reports
- ``docs``: Sphinx documentation

For a live view, see:
https://github.com/EdAlita/ChameleonIQ/wiki/Architecture
