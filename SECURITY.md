# Security Policy

## Supported Versions

Only the latest non-prerelease version is supported

## Security Contact and Vulnerability Reporting

If you discover a security vulnerability in this project, please contact me **directly** at:

**Email:** eulinbriseo@gmail.com  
**Subject line:** [SECURITY ALERT]

---

### What to expect after reporting

- You will receive an initial acknowledgment **within one week** of your report.
- Updates on the investigation and resolution progress will be provided **at least once every two weeks** until the issue is resolved or declined.
- If the vulnerability is accepted, a fix or mitigation plan will be developed and a coordinated disclosure will be arranged.
- If the report is declined (e.g., not a security issue or already resolved), you will receive an explanation.

---

Thank you for helping keep this project safe and secure!
